# zero
**Write machine learning algorithms from Zero**

## Features
- Friendly code.
- Implementation from Zero.
- Designed for easily learning algorithms.

## Getting started
Linear regression demo.
```python
from zero.api import LinearRegression
model = LinearRegression()
model.fit(X, y)
```
