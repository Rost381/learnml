Covariance Matrix

"协方差矩阵" in Chinese.

Covariance matrix is a matrix whose element in the i, j position is the covariance between the i-th and j-th elements of a random vector.

Definition

If the entries in the column vector

X=(X1,X2,...,Xn)T {\mathbf {X} =(X_{1},X_{2},...,X_{n})^{\mathrm {T} }} X=(X1​,X2​,...,Xn​)T

are random variables, each with finite variance and expected value, then the covariance matrix K⁡XX{\operatorname {K} _{\mathbf {X} \mathbf {X} }}KXX​ is the matrix whose (i,j){(i,j)}(i,j) entry is the covariance:

K⁡XiXj=cov⁡[Xi,Xj]=E⁡[(Xi−E⁡[Xi])(Xj−E⁡[Xj])]{\operatorname {K} _{X_{i}X_{j}}=\operatorname {cov} [X_{i},X_{j}]=\operatorname {E} [(X_{i}-\operatorname {E} [X_{i}])(X_{j}-\operatorname {E} [X_{j}])]}KXi​Xj​​=cov[Xi​,Xj​]=E[(Xi​−E[Xi​])(Xj​−E[Xj​])]

Estimation

If MX{\mathbf {M} _{\mathbf {X} }}MX​ and MY{\mathbf {M} _{\mathbf {Y} }}MY​ are centred data matrices of dimension n×p{n\times p}n×p and n×q{n\times q}n×q respectively, i.e. with n rows of observations of p and q columns of variables, from which the column means have been subtracted, then, if the column means were estimated from the data, sample correlation matrices QX{\mathbf {Q} _{\mathbf {X} }}QX​ and QXY{\mathbf {Q} _{\mathbf {XY} }}QXY​ can be defined to be

QX=1n−1MXTMX,QXY=1n−1MXTMY{\mathbf {Q} _{\mathbf {X} }={\frac {1}{n-1}}\mathbf {M} _{\mathbf {X} }^{\rm {T}}\mathbf {M} _{\mathbf {X} },\qquad \mathbf {Q} _{\mathbf {XY} }={\frac {1}{n-1}}\mathbf {M} _{\mathbf {X} }^{\rm {T}}\mathbf {M} _{\mathbf {Y} }}QX​=n−11​MXT​MX​,QXY​=n−11​MXT​MY​

def covariance_matrix(X, Y=None):
    if Y is None:
        Y = X
    n_samples = np.shape(X)[0]
    covariance_matrix = (1 / (n_samples - 1)) * \
        (X - X.mean(axis=0)).T.dot(Y - Y.mean(axis=0))
    return np.array(covariance_matrix, dtype=float)
